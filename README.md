SmoredBoard Updater

Update.txt is the main file the SmoredBoard reads to see if there is a new update.

We get changes now from the lines "1.0.4.1=raw.github...." at the bottom. 

We no longer use the changes= BUT you still need to populate that for 
OLD version of the SmoredBoard (1.0.4.1<=) that don't have the new "smart" updater
